# Archives du dashboard 2025

Ce dossier conserve les analyses, exports et outils exploratoires produits autour du premier
dashboard en 2025.

Ils sont **historiques et non canoniques** :

- les définitions et seuils peuvent différer du glossaire 2026 ;
- certains scripts supposent l'ancienne arborescence du dépôt Next.js ;
- leurs chiffres ne doivent pas être cités comme résultats actuels.

Pour toute analyse courante, utiliser `enquete_usages_2026/`, ses tables canoniques et ses
`facts_*.json`.
